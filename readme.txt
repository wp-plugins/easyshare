== Description ==

easyShare is a plugin to share your content with friends, social networks etc ...
It is fully customizable : icons path, position, display mode, bloc title, bookmark text, custom css and class, servies to display.
And yeah, there is a "test mode" in the options to configure your plugin when you edit your posts

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Optionnal, enable the "test mode" to configure your plugin easily
